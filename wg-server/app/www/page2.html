Manage Clients via CSV

Download Client Data

Click the button below to download the current list of clients as a clients.csv file. You can then edit this file to add new clients and upload it again.

Upload Client Data

Select a .csv file to upload. The file must have a header row with columns: name,username,password. The system will create a new WireGuard client for each name in the file. Existing clients will not be affected. The username and password columns are reserved for future use.
